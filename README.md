## Quickstart
To get started with FieldportX1, follow these steps:
1. Clone the repository:
   ```bash
   git clone https://github.com/scismjacob38-cmyk/FieldportX1.git
   cd FieldportX1
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the application:
   ```bash
   npm start
   ```
